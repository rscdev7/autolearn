DomainEvent¶

@author : rscalia

@build-date : Sun 25/07/2021

@last-update : Sun 25/07/2021

Questa classe mantiene le informazioni correlate al verificarsi di un evento nel sistema.

class catalog.lib.abstract_event_sourcing.DomainEvent.DomainEvent(pEntityId: str, pEventTimeStamp: int, pEventPayload: dict)¶

Basi: object

__init__(pEntityId: str, pEventTimeStamp: int, pEventPayload: dict) → object¶

Costruttore, permette di raccogliere le informazioni necessarie all’identificazione univoca dell’Evento.

Parametri

  • pEntityId (str) – entità di dominio che ha subito l’evento

  • pEventTimeStamp (int) – timestamp dell’evento

  • pEventPayload (dict) –

    informazioni necessarie a identificare il cambiamento scaturito dall’evento

    Formato Dizionario:

    • »key»: bytes

    • »value»:

      • dict { … «payload»: _ … }

    • »timestamp»: int
