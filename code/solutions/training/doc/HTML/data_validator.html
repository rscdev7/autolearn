TrainRequestValidator¶

@author : rscalia

@build-date : Thu 03/08/2021

@last-update : Thu 03/08/2021

Questo componente serve per validare una richiesta di training del Cliente

class training.lib.data_validator.TrainRequestValidator.TrainRequestValidator¶

Basi: object

static checkHyperparams(pReqHyperParams: List[dict], pCatalogHyperParams: List[dict], pModelTaskPair: tuple) → Union[bool, Tuple[bool, str], Exception]¶

Questa funzione permette di controllare se una lista di Iperparametri è conforme rispetto al catalogo.

Parametri

  • pReqHyperParams (List[dict]) – Iperparametri Richiesta

  • pCatalogHyperParams (List[dict]) – Iperparametri Catalogo abbinati alla richiestqa

  • pModelTaskPair (tuple) – Coppia Modello-Task scelti

Ritorna

True se la richiesta è corretta; altrimenti (FALSE , MSG_ERRORE) oppure Eccezione se il formato della richiesta/catalogo è diverso rispetto a quello sopra elencato.

Tipo di ritorno

Union[ bool , Tuple[bool , str ] , Exception ]

Solleva

Exception – eccezione derivata dall’assenza di una chiave nel dizionario
