@author : rscalia @build-date : Thu 15/07/2021 @last_update : Fri 23/07/2021

Questa classe serve per gestire le chiamate HTTP Sincrone e Asincrone

get(pURL: str, pParams: dict = {}) → Union[Dict[int, dict], Exception, Tuple[Exception, int]]¶

Questo metodo effettua una get Sincrona.

Parametri

  • pURL (str) – URL su cui fare la chiamata

  • pParams (dict | DEF = {}) – eventuali parametri di query da aggiugere all’URL http

Ritorna

dizionario contenente status_code e payload restituiti dal server

Formato Dizionario:

  • status_code, status code della chiamata HTTP

  • payload, dati restituiti dal server

Tipo di ritorno

(Union[ Dict[int,dict] , Exception, Tuple [Exception , int] ])

Solleva

  • "HTTPError" – errore HTTP con codice da 400 a 499

  • "RuntimeError" – errore generico a Runtime

  • "Exception" – errore generico

async getAsync(pURL: str, pParams: dict = {}) → Union[Dict[int, dict], Exception, Tuple[Exception, int]]¶

Questo metodo effettua una get asincrona.

Parametri

  • pParams (dict | DEF = {}) – eventuali parametri di query da aggiugere all’URL http

Ritorna

dizionario contenente status_code e payload restituiti dal server

Formato Dizionario:

  • status_code, status code della chiamata HTTP

  • payload, dati restituiti dal server

Tipo di ritorno

(Union[ Dict[int,dict] , Exception, Tuple [Exception , int] ])

Solleva

  • "HTTPError" – errore HTTP con codice da 400 a 499

  • "InvalidURL" – URL mal formato, ad esempio manca l’host

  • "ClientPayloadError" – il client ha inviato al server dei dati mal formati

  • "ClientError" – errore generico al livello di API HTTP

  • "RuntimeError" – errore generico a runtime

  • "Exception" – errore generico
